= WebTestClient Sample

A basic application that demonstrates using Spring REST Docs with Spring WebFlux's `WebTestClient`.



== Building the Sample

You will need Java 17 to build the sample.

=== Notes for first-time-users

You can verify that Java 17 is being used by running:

[source,shell]
----
java -version
----

It is built using Gradle:

[source,shell]
----
./gradlew build
----

Running the build executes the sample's tests, which generate Spring REST Docs snippets, and then 
uses Asciidoctor to assemble the final API documentation.

As part of the build, a file named `build/docs/asciidoc/index.html` is created. Open this file 
in a browser to view the generated documentation.

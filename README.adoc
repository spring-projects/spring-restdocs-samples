= Spring REST Docs Samples



== Overview
This repository contains a number of samples for https://github.com/spring-projects/spring-restdocs[Spring REST Docs] 4.0 built using Spring Boot 4.0.
For samples using Spring REST Docs 3.0 and Spring Boot 3.5, please see the https://github.com/spring-projects/spring-restdocs-samples/tree/3.0.x[3.0.x branch].
